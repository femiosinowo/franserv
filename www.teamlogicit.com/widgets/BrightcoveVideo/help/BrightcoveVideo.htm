Brightcove Video Widget

Plays any Brightcove video on a PageBuilder Page. Depending on the features of your Brightcove account, videos may be encoded automatically into various formats such as h.264 or VP6 (flv).

 

Note: Before you can use this widget, you must have an account with http://www.brightcove.com.

Configure the Brightcove Video widget in the Ektron CMS400.NET Workarea > Settings > Personalizations > Brightcove widget.

To change the properties, click Edit ().

Properties

 
Property Value
Widget Title Displays as <alt> text when you mouse-over the widget in the widget tray. Default value is BrightcoveVideo.
Widget Label This text is shown as the label on the bottom of the BrightcoveVideo widget icon. Default value is BrightcoveVideo.
Show Setup (boolean) Set this to true to show the account setup screen when you edit the widget configuration.
PlayerID (String) The Brightcove Player. See your Brightcove account for player configurations you may use.
PublisherID (Int64) The ID given to your account from Brightcove.
token (String) A value given to you by Brightcove giving you the ability to view videos.
writetoken (String) A value given to you by Brightcove giving you the ability to post new videos through this widget.

Click Edit () to select the video and player to use in this widget after configuring the Brightcove account information.
